GAO ZE QIAN / GLASS PROCESS

Glass Object Study

该作品由AI视觉生成（Midjourney + 混元模型）开始， 在Cinema4D中重构三维结构，并通过3D打印进入现实材料系统。 石膏翻模 / 硅胶模具 / 蜡模修型 / 高温石膏铸造， 最终在玻璃窑中完成物理转译。

Material Pipeline

AI生成 → C4D建模 → 3D打印 → 翻模系统 → 蜡模精修 → 耐火石膏 → 窑炉玻璃铸造。 这是一个从“数字生成”到“物理凝固”的完整链路。
