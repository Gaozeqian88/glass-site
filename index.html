Material Translation

该作品源自Midjourney生成图像与混元AI生成模型， 并在Cinema4D中进行结构重建。 最终通过3D打印、石膏翻模、硅胶开模、蜡模制作、 耐火石膏铸型与高温玻璃熔制完成实体转化。 这一过程不仅是造型的实现，更是材料从数字到物理的连续生成。

Production Pipeline

Midjourney视觉生成 → 混元AI结构建模 → Cinema4D拓扑优化 → 3D打印原型 → 石膏/硅胶翻模 → 蜡模精修 → 耐火石膏模 → 高温窑炉玻璃铸造。 每一步都不是“制作”，而是材料状态的转换。
