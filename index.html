GAO ZE QIAN / GLASS PROCESS

Glass Object Study

该作品基于AI生成视觉（Midjourney + 混元模型）， 在Cinema4D中进行结构重建， 并通过3D打印进入真实材料转译流程。 石膏翻模 / 硅胶开模 / 蜡模修正 / 耐火石膏铸型 / 高温玻璃熔制， 最终形成玻璃实体作品。

Material Pipeline

AI生成 → C4D建模 → 3D打印原型 → 石膏/硅胶翻模 → 蜡模修型 → 耐火石膏 → 窑炉玻璃铸造。 这是一个“数字 → 物理”的连续生成系统。
